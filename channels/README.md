# Channels

This directory is reserved for realized environment pointers or manifests.

Shared channel meaning is defined in `SocioProphet/socioprophet-agent-standards/schemas/channel.schema.json`.

Initial channel set:

- `dev`
- `candidate`
- `stable`

This repository should realize those channel surfaces for Linux hosts and images, while `SocioProphet/agentplane` continues to define the control-plane promotion model.
