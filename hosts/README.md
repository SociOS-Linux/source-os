# Hosts

This directory carries concrete machine-role definitions for SourceOS Linux realization.

Planned initial roles:

- `builder-aarch64/`
- `canary-x86_64/`
- `stable-x86_64/`

Control-plane placement, promotion, and evidence semantics are defined in `SocioProphet/agentplane`.
Shared channel and capability vocabulary are defined in `SocioProphet/socioprophet-agent-standards`.
